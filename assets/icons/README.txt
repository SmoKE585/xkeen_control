Icon files for VPN_TRAY

Put files in this folder with exact names:
- app.ico            (for EXE + window icon)
- tray_on.png        (tray icon when VPN is ON)
- tray_off.png       (tray icon when VPN is OFF)
- tray_unknown.png   (tray icon when status is UNKNOWN)

Recommended sizes:
- app.ico: multi-size ICO with 16, 24, 32, 48, 64, 128, 256 px layers.
- tray_*.png: 64x64 px RGBA with transparent background.

SVG note:
- You can design in SVG, but convert before use.
- EXE icon requires .ico.
- Tray icons should be .png (or .ico), not raw .svg.
